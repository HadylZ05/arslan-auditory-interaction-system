# Auditory System Screenshots

This directory contains visual references related to the **Auditory Interaction System**
of the Arslan humanoid robot.

The screenshots are provided to illustrate system behavior and user interaction
without exposing source code or proprietary dependencies.

---

## Included Screenshots

- User interface prototype for the auditory interaction system
- Console output demonstrating speech recognition and response handling
- Interaction examples during system testing

---

## Notes

- The implementation relies on external libraries and DLLs that are not included
  in this repository.
- Screenshots are provided strictly for documentation and demonstration purposes.
